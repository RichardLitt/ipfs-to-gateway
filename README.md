# ipfs-to-gateway

> Automatically add and open a file in the ipfs.io gateway

## Install

```
npm i -g ipfs-to-gateway
```

## Usage

I aliased this to ipfs-io because it seems easier and more intuitive to me than ipfs-to-gateway.

```sh
$ ipfs-io <file>
```
